# Unofficial stress-ng ARMHF static build

Unofficial, unmodified build of [stress-ng](https://github.com/ColinIanKing/stress-ng)
by Colin Ian King.

- Upstream: https://github.com/ColinIanKing/stress-ng
- License: GPL-2.0
- Target: ARMv8-A Cortex-A53, AArch32 / ARMHF
- Linking: static
- Built from: [V0.21.03](https://github.com/ColinIanKing/stress-ng/tree/V0.21.03)

Build flags:
-mcpu=cortex-a53 -marm -mfloat-abi=hard

This repository only distributes a build artifact for personal device use.
